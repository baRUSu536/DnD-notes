# Погода

| к4  | Погода                                                                                         |
| --- | ---------------------------------------------------------------------------------------------- |
| 1   | **Ужасная**. Подъём возможен, только если ты сверхчеловек, для простых смертных это невозможно |
| 2   | **Плохая**. Сильные помехи, из-за которых очень тяжело продвигаться вперёд                     |
| 3   | **Средняя**. Небольшие неприятности, влияющие на скорость подъёма                              |
| 4   | **Хорошая**. Не мешает походу и не доставляет помех                                            |
Длительность погоды равна **1к4** часов.
## Хорошая погода 

| к6  | Погодные условия                                                                                                                                                                                                          |
| :-: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  1  | **Скользкий наст.**<br>Местность считается **труднопроходимой** для существ без подходящей обуви (шипы, кошки) или без знания местности. Проверка *Ловкости (Акробатика)* со Сл 10 при быстром движении, чтобы не упасть. |
|  2  | **Легкий снегопад.**<br>Максимальная видимость 500 футов. Для просмотра на дальнюю дистанцию требуется проверка *Мудрости (Внимательность)* с Сл 15.<br>*Направление ветра бросать по таблице "Направление".*             |
|  3  | **Легкий дождь.** Неприятно, но не критично.<br>Если находится долго под дождём, снаряжение намокнет.                                                                                                                     |
|  4  | **Переменная облачность / облачность.**<br>Небо наполовину или полностью покрыто облаками. Может быть немного уныло и темновато, но проблем не доставляет.                                                                |
|  5  | **Лёгкий туман**. Видимость слегка ухудшена.<br>Местность дальше 150 фт. считается слабо заслонённой.                                                                                                                     |
|  6  | **Полный штиль / слабый ветер.** Ясная погода, которую прерывает почти неощутимый ветерок.<br>*Направление ветра бросать по таблице "Направление".*                                                                       |

## Средняя погода
| к6  | Погодные условия                                                                                                                                                                                                                                                                                                                   |
| :-: | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  1  | **Слякоть**. Днём солнце подтапливает снег, превращая тропы в холодную жижу. Снег липнет к обуви, делая каждый шаг тяжелым.<br>Скорость передвижения уменьшена на 10 футов. **Проверка Ловкости (Акробатика) или Силы (Атлетика) (Сложность 11)** при перемещении по склону, чтобы не упасть и не скатиться вниз на 1к6 x 5 футов. |
|  2  | **Снегопад.** Видимость ухудшена.<br>Максимальная видимость 200 футов. Для просмотра на дальнюю дистанцию требуется проверка *Мудрости (Внимательность)* с Сл 15.                                                                                                                                                                  |
|  3  | **Град.** Довольно крупный град неприятно колотит вас.<br>Каждый час, пока не закончится град, требуется проходить спасбросок **Силы** или **Телосложения** с Сл 10. При провале вы получаете 1 единицу *дробящего* урона.                                                                                                         |
|  4  | **Пасмурно.** Сильно сокращена видимость.<br>Максимальная видимость 120 футов.<br>Помеха на все проверки Выживания для ориентирования.<br>Существам сложнее определить точное время суток без часов.                                                                                                                               |
|  5  | **Туман**. Вся местность покрывается туманом.<br>Местность дальше 120 фт. считается сильно заслонённой.                                                                                                                                                                                                                            |
|  6  | **Сильный ветер.** Этот ветер может как затруднять, так и упрощать перемещение.<br>Нахождение более 60 минут требует спасбросок **Сопротивления морозу**.<br>*Направление бросать по таблице "Направление".*                                                                                                                       |
## Плохая погода
В такую погоду проходить спасбросок **Сопротивления морозу** нужно вдвое чаще, а небо всегда заслонено тучами. Погода считается пасмурной.

| к6  | Погодные условия                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| :-: | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  1  | **Снежная доска**. Под тонким твердым настом лежит рыхлый слой снега. Снежная доска может треснуть и соскользнуть под ногами.<br>Передвижение требует проверки *Мудрости (Выживание)* со Сл 16, чтобы выбрать безопасный маршрут. Каждый крупный или шумный персонаж (в тяжелой броне) должен совершить проверку Ловкости со Сл 14. Двойной провал означает спуск локальной снежной доски: все существа в радиусе 20 футов должны сделать спасбросок Ловкости со Сл 14. Провал — существо сносится вниз на 4к8 x 5 футов, получает 2к6 дробящего урона и зарывается в снег (состояние "Лежащий ничком" и "Опутанный"). |
|  2  | **Снежная буря**. <br>Эффект грозы.<br>Эффект шторма.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|  3  | **Сильный град.** Мощный град может оставить вам парочку синяков.<br>Каждые 10 минут, пока не закончится град, требуется проходить спасбросок **Силы**, **Ловкости** или **Телосложения** с Сл 13. При провале вы получаете 1 единицу *дробящего* урона.                                                                                                                                                                                                                                                                                                                                                               |
|  4  | **Гроза.** Снегопад с грозой<br>Начинается событие [[Гроза]].<br>Максимальная видимость 120 футов. Для просмотра на дальнюю дистанцию требуется проверка *Мудрости (Внимательность)* с Сл 15 с **помехой**.<br>Каждые **5к4** минут молния ударяет в землю.                                                                                                                                                                                                                                                                                                                                                            |
|  5  | **Густой туман.** Не видно не зги.<br>Дальше 15 футов местность считается сильно заслонённой.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|  6  | **Шторм.** В такой ветер невозможно нормально перемещаться.<br>Вне зависимости от направления скорость снижена на 10 футов.<br>*Направление ветра бросать по таблице "Направление".*                                                                                                                                                                                                                                                                                                                                                                                                                                   |
## Ужасная погода
В такую погоду проходить спасбросок **Сопротивления морозу** нужно в четыре раза чаще, а небо всегда заслонено тучами.

| к6  | Погодные условия                                                                                                                                                                                                                                                                                                                                                          |
| :-: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  1  | **Лавина.** У вас есть считанные секунды, чтобы укрыться от лавины.<br>Начинается событие [[Лавина]].<br>После лавины начинается снежная буря.                                                                                                                                                                                                                            |
|  2  | **Буран.** Сильнейший ураганный ветер со снегопадом, застилающим обзор.<br>*Максимальная видимость* 60 футов. Всё, находящееся дальше этой дистанции, считается *сильно заслонённым*.<br>*Максимальная слышимость* 10 футов. Дальше этой дистанции ничего не слышно.<br>*Направление ветра бросать по таблице "Направление".*                                             |
|  3  | **Смертоносный град.** Градины, размером с кулак того и гляди пришибут, а из-за сильного ветра сражаться с градом становится совершенно невозможно.<br>Эффект шторма.<br>Единственные укрытия — это деревья с густой и широкой кроной, пещеры и ущелья, а также постройки из твёрдых материалов.<br>Каждую минуту нахождения под градом вы получаете 1к4 дробящего урона. |
|  4  | **Сильная гроза.** <br>Эффект бурана.<br>Начинается событие [[Гроза]].<br>Каждые **1к6** минут молния ударяет в землю. Молния всегда бьёт по таблице "Удар молнии".                                                                                                                                                                                                       |
|  5  | **Удушающий туман**. Это не просто вода, а магическая или ядовитая испарина, которая не только скрывает, но и вредит.<br>Начинается событие [[Удушающий туман]].                                                                                                                                                                                                          |
|  6  | **Буран.** Сильнейший ураганный ветер со снегопадом, застилающим обзор.<br>*Максимальная видимость* 60 футов. Всё, находящееся дальше этой дистанции, считается *сильно заслонённым*.<br>*Максимальная слышимость* 10 футов. Дальше этой дистанции ничего не слышно.<br>*Направление ветра бросать по таблице "Направление".*                                             |
