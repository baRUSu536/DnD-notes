# Внешность
Пвю Сиэн Доехт - массивное существо, похожее на орла, полностью состоящее из кристаллов голубого льда. У нее также есть единственный кристалл цвета фуксии на лбу. В форме яйца она представляет собой ледяной шар, который светится белой магией и кажется окутанным гнездом из ледяных шипов.
