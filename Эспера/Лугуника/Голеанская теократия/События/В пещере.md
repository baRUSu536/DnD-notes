Определить тип пещеры по следующей таблице:

| к20   | Вид пещеры                                                                   |
| ----- | ---------------------------------------------------------------------------- |
| 1-3   | Неглубокая берлога бурого медведя ([[Бурый медведь]]).                       |
| 4-8   | Длинная извилистая пещера, берлога пещерного медведя ([[Пещерный медведь]]). |
| 9     | Пещера, уходящая глубоко внутрь горы.                                        |
| 10-13 | Пещера, проходящая насквозь всей горы.                                       |
| 14-16 | Пещера, уходящая под землю.                                                  |
| 17-20 | Небольшая пещера.                                                            |

| к20   | Описание                                    |
| ----- | ------------------------------------------- |
| 1-5   | Нет ничего, кроме голых камней и снега      |

| 11-17 | Есть съедобные грибы.                       |
| 18-19 | Логово грибов погибели ([[Гриб погибели]]). |
| 20    | Есть магические грибы                       |
