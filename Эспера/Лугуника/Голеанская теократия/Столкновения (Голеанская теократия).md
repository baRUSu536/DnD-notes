#дополнительные_правила #события #таблицы
___
Броски по таблицам столкновений производятся раз в 2 часа, при условии перемещения. Если герои устраивают ночлег в безопасном месте, то события происходят в стороне.
# Первый уровень (высота 3000 – 6000 фт.)

| к100     | Событие                                                                                                                                                              |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1-20<br> | Ничего не происходит.<br>Сделать бросок по таблице Хорошая погода.                                                                                                   |
| 21-30    | Вы натыкаетесь на следы зайца. Совершив проверку Мудрости (Внимательность) вы обнаруживаете заячью нору. Внутри **1к4** взрослых особей и **3к4 - 3** молодые особи. |
| 31-35    | Мимо вас пробежала белка. Возможно она бежит в своё дупло.                                                                                                           |
| 36-40    | На вас напали **1к4** бешеных белок ([[Белка (бешеная)]]).                                                                                                           |
| 41-50    | Вы заметили, как перед вами пролетела сова.                                                                                                                          |
| 51-70    | Вы натыкаетесь на большое стадо козлов ([[Козёл]]) в количестве **4к6 + 4** особей                                                                                   |
| 71-89    | Вы услышали крик орла над вами.                                                                                                                                      |
| 90-96    | Вы натыкаетесь на стаю волков ([[Волк]]) в количестве **1к4** особей                                                                                                 |
| 97-98    | Вы натыкаетесь на бурого медведя ([[Бурый медведь]])                                                                                                                 |
| 99-100   | Сделать три раза бросок по этой таблице. Если выпадает значение 1-20 или 99-100, пропустить его.                                                                     |
